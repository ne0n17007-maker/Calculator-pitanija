# NutriCalc — Калькулятор питания и БЖУ

Простой веб-сайт для расчёта дневной нормы калорий и баланса белков, жиров и углеводов (БЖУ).

## Возможности

- Приветственный лендинг с описанием сервиса
- Калькулятор калорий на основе формулы **Mifflin-St Jeor**
- Учёт пола, возраста, веса, роста и уровня активности
- Корректировка под цель: похудение, поддержание веса, набор массы
- Расчёт рекомендуемого БЖУ на день
- Адаптивный дизайн для мобильных устройств

## Запуск

Проект состоит из статических HTML-файлов — сервер не обязателен.

1. Откройте `index.html` в браузере двойным щелчком
2. Или запустите локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Затем откройте в браузере: `http://localhost:8000`

## Как работает расчёт

1. **BMR** (базовый обмен) — формула Mifflin-St Jeor
2. **TDEE** — BMR × коэффициент активности
3. **Корректировка цели:**
   - Похудение: −20%
   - Поддержание: без изменений
   - Набор массы: +15%
4. **БЖУ:**
   - Белки: 2 г × вес (кг)
   - Жиры: 0.9 г × вес (кг)
   - Углеводы: остаток калорий
